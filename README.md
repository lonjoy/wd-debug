# 前端调试工具


### 课程简介

介绍前端常用/不常用调试工具，包括浏览器控制台，插件，请求监测工具，兼容性测试工具等。

### 授课目标

* 介绍常用调试工具
* 讲解调试工具的使用
* 常见问题的调试方法
